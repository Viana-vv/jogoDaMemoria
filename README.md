# # Jogo da Memória

Um jogo da memória desenvolvido em **HTML**, **CSS** e **JavaScript**, utilizando imagens de frutas.  
O objetivo é encontrar todos os pares antes de perder.

## Funcionalidades
- Tabuleiro gerado dinamicamente com cartas embaralhadas.
- As cartas são reveladas ao clicar e voltam para o verso se não formarem par.
- Mensagem de vitória exibida quando todos os pares são encontrados.
- Aviso de derrota exibido quando o jogador erra um par.
- Reinício automático após derrota.
- Botão para reiniciar manualmente o jogo.

## Estrutura do Projeto
jogoDaMemoria/
│── index.html        # Estrutura principal do jogo
│── style.css         # Estilos e layout
│── script.js         # Lógica do jogo
│── img/              # Imagens das cartas e ícones


## Como Jogar
1. Abra o arquivo `index.html` em um navegador.
2. Memorize a posição das cartas antes delas virarem para o verso.
3. Clique em duas cartas para tentar formar um par.
4. Se acertar, as cartas permanecem viradas.
5. Se errar, aparece o aviso de derrota e o jogo reinicia.
6. Continue até encontrar todos os pares e vencer.
7. Use o botão **Reiniciar Jogo** para começar novamente a qualquer momento.
